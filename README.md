# Vulkan Teapots

Hello vulkan! Learner project in C. Implements a well behaved dynamic Vulkan renderer, and spins some lightly shaded teapots around.

## Building
### Linux 
```
./clear.sh
./build.sh 
./run.sh
```

## Libraries
Place them in lib folder of this project.
- glfw
- cglm
- VulkanMemoryAllocator
- fast_obj
- stb/stb_image.h
